# Week 1 - portfolio

## Project Overview
This project is a simple Personal Portfolio Website built using only HTML5.  
It includes multiple sections such as About, Skills, Projects, and Contact, along with internal navigation links.

## Technologies Used
- HTML5 (Semantic HTML)

## Folder Structure
week1-portfolio/
├─ index.html
├─ README.md
└─ images/
   └─ profile.jpg

## How to Run the Project (Local Setup)
1. Download or clone the project folder.
2. Open the `week1-portfolio` folder in VS Code.
3. Open `index.html` in a browser:
   - Option 1: Right-click `index.html` → Open with Live Server (recommended)
   - Option 2: Double click `index.html` to open in Chrome/Edge

## Features Implemented
- Semantic page structure: header, nav, main, sections, footer
- Internal navigation menu using anchor links
- About section with profile image
- Skills list
- Projects section with external links
- Contact form with name, email and message fields

## Notes / Customization
- Replace `images/profile.jpg` with your own photo (keep the file name same or update the HTML path).
- Replace project links with your actual GitHub or demo links.
- Update the email address in the Contact section.

## Validation (Optional but Recommended)
You can validate the HTML using W3C Validator:
https://validator.w3.org/
Upload or paste your HTML code there to check for errors.
